***PAGINA DE LA GALERIA DE TRIPLE PEAKS***

Originalmente llamado "proyecto homeland", la pagina de la galeria triple peaks sigue un diseno original enviado por la plataforma figma, se creo en html y CSS con una fuente "inter" incorporada al proyecto, flexbox, y diseÂ¤o responsivo para visualizarlo en 3 diferentes tipos de pantalla: movil (320px) con un breakpoint a los 544px; tablet (768), entre 545 y 1024px; y desktop para pantallas con resoluciones superiores. La pagina contiene varias secciones entre las que se incluyen un foto grid y una seccion de articulos breves sobre diversos lugares, en donde se incluye una imagen y la opcion de comprar la misma en NFT.
